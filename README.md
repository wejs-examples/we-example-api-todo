# we-example-api-todo
> A todolist example using We.js resource API

This is an example how to use We.js resource API.

This example will be consumed by: 
    - React.js
    - Angular.js
    - Ember.js
    
### How to test

After clone and install npm packages:

```sh
npm test
```
